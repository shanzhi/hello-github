# 任督二脉

1. 本地创建文件夹                                                              （a）
2. git clone htpps://...                                                       (b)  pull
3. 添加要上传的文件  到   拉取来的文件夹                                         （c.txt）
4. git add .
5. git commit -m "first commit"
6. git push -u origin main                                                      (若fatal,请再try)
